Pinboard Project

This application allows people to create a virtual pinboard for images and photos that they find online. It will be a similar but more basic experience to the popular website Pinterest

Goals:

Visitors should be allowed to:

  • open a simple static About page, that includes the title of the project option, a brief description of what the website is for and who built it.
  • register and log in as a User with their email using a basic sign up and log in form.
  • to view the six most-recently created Pins on the homepage.
  • to search for Pins with a search term that will match on either title or tag from a form on the homepage.
  • to view all of the details for a Pin plus all Comments related to that Pin on one view.
  • to have a link to the homepage, the about page, and the login/register view in the header of every view.

And the Users (registred visitors) should be allowed to:

  • to create Pins that include a title, an url for the image that they want to “pin", and a tag, which is a phrase or category relevant to the Pin.
  • to edit all details of a Pin by clicking on a link from the “Show pin” view.
  • to save new or updated Pins only if a title is provided, and that a tag is no longer than 30 characters in length, although a tag is optional. Any validation errors should be displayed helpfully for User
  • to add a comment on a “show pin” view when they are signed in
  • to add any Pin created by any User to their "personal Pin list" from the show Pin view.
  • to have a link to the homepage, the about page, and their personal pin list in the header of every view.
